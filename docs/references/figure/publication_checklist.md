# Publication Figure Checklist

Use this checklist before exporting final figures.

## Content

- The figure answers one clear question.
- Sample sizes are visible or stated.
- Missing data handling is stated when relevant.
- Outliers are shown, annotated, or removed with a documented reason.
- Statistical annotations match the tested comparison.

## Design

- Axes have units.
- Labels are readable at final size.
- Legend has six or fewer items, or the figure is split.
- Colors are colorblind-safe.
- Grayscale print remains interpretable when needed.
- No 3D effects or decorative chart junk.

## Export

- Size matches the target venue.
- Font size matches the target venue.
- Vector output is used when possible.
- Raster output meets DPI requirements.
- File name and panel labels match the manuscript.

This checklist does not replace journal instructions or statistical review.
